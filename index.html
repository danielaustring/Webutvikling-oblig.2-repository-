Git/Github:

Github tar føre seg en veldig viktig rolle i dataverdenen når det kommer til dataprogrammering. Det gjør det lett for deg som bruker og å arbeide sammen med andre om prosjekt fra hvor som helst, når som helst. Du får lagret alle slags filer. Om det er bilder, videoer eller generelle kodelinjer. «Repository», som da er din egen lagringsplass. Her finner du alt du har lagret, du kan til og med dele og diskutere ressursene dine med andre.

Trykk for å les mer!

Media Typer:

Når man lager nettsider finnes det flere måter å bruke nettsiden på en kun å sjå den gjennom skjermen. Ved å bruke noe kalt media typer, kan man få flere brukere til å lese eller ta til seg innholdet du har skrevet. Disse mediene er målretta mot for eksempel blinde folk som kan bruke tale og print sånn man kan printe ut hele siden i ordformat. De forskjellige media typene vi har er: all, print, screen, speech, aural, braille, handheld, projection, tty og tv

Trykk for å les mer!

CSS Layouts:

Når det kommer til Css layouts har vi tre forskjellige metoder som er brukt. De er Css Grid, etterfulgt av Css flexbox og til slutt har Css float. Når vi skal lage en nettside må vi ha en plan for hvordan nettstedet ditt skal bli, og derfor er det viktig å vite hvordan du skal stile den. Ved å bruke disse metodene vil du få ei god og strukturert nettside, noen litt lettere enn de andre.

Trykk for å les mer!
